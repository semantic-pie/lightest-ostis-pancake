# Lightest OSTIS Pancake (dockerised)

## Install 

Clone repository:

```sh
git clone https://github.com/semantic-pie/lightest-ostis-pancake
```

install ostis components:

```sh
cd lightest-ostis-pancake
./pancake.sh install
```

that's it!

## Run

This `Lightest OSTIS Pancake` runs only in docker:

```sh
docker-compose up
```
